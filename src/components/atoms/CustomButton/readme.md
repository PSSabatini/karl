# Custom Button

Available color classes:

- neutral (default)
- black
- green
- orange
- yellow
- brown

## Usage Template
```html
    <button class="custom-button [color] [--invert] [--border]" [disabled]>[label]</button>
```

## Usage Vue
```html
    <custom-button :color="color" :invert="--invert" :border="--border" :label="label" :disabled="disabled"></custom-button>
```
