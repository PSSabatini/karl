# Button With Icon

Button that is possible to add an icon.

Available Icons:
  - applepay
  - cart-plus
  - chevron_left
  - paypal

Available Colors: 
  - neutral (default)
  - black
  - yellow
  - brown
  - green
  - orange

Available positions (in relation of the lable):
  - left
  - right

## Usage

```html
  <button-with-icon 
    iconOption="[ String ]" 
    color="[ String ]" 
    position="[ String ]" 
    label="[ String ]">
  </button-with-icon>
```
